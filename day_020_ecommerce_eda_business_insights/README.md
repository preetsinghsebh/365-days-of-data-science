## Project Overview
Exploratory Data Analysis (EDA) on an e-commerce customer behavior dataset to understand
sales trends, category performance, and customer contribution.

## Dataset
E-Commerce Customer Behavior & Sales Analysis (Kaggle)

## Key Analysis Performed
- Sales distribution across product categories
- Revenue contribution by category
- Order count by category
- Customer spending distribution
- Identification of high-value customers

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- A small number of categories contribute most of the revenue
- Customer spending is skewed (few high-value customers)
- Certain categories generate high order volume but lower revenue

## Output
- Cleaned dataset (`ecommerce_eda.csv`)
- EDA notebook with visualizations and insights
