## Day 53 – Model Comparison (NLP)

Today I compared two models for sentiment analysis using the IMDB dataset.

Models used:
- Logistic Regression
- Naive Bayes

Steps:
- Converted text to numerical features using TF-IDF
- Trained both models on the same data
- Compared their accuracy

Result:
Logistic Regression performed better than Naive Bayes.

Learning:
Model selection impacts performance even with the same data.
